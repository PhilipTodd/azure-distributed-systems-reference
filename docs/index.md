# Distributed Systems Reference Platform

A production-style reference implementation demonstrating modern distributed systems architecture using **.NET 10** and **Microsoft Azure**.

The project is being developed to showcase cloud-native engineering practices commonly used in enterprise software, including:

- Microservices
- Event-Driven Architecture
- API Gateway
- Asynchronous Messaging
- Azure Service Bus
- Microsoft Entra ID
- Azure App Service
- Infrastructure as Code
- Azure DevOps CI/CD
- Observability
